Lịch sử

  • x² - 10x + 16 = 0, với hai nghiệm phân biệt x₁ = 2, x₂ = 8
  • x² - 6x - 40 = 0, với hai nghiệm phân biệt x₁ = -4, x₂ = -10
  • x² - 8x + 16 = 0, với nghiệm kép x = 4
  • x² + x + 1 = 0, phương trình vô nghiệm thực
  • 0x² + 0x = 0, phương trình có vô số nghiệm
